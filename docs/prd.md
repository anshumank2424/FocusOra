# FOCUSORA PRD

FocusOra is a study focus application designed for students preparing for competitive exams.

Target Audience:
- JEE
- NEET
- UPSC
- College Students

Primary Goal:
Help students reduce distractions and increase deep work hours.

Core Features:
- Pomodoro Timer
- Stopwatch
- Focus Timer
- App Blocking
- Analytics
- XP System
- Streaks
- Achievements

Tech Stack:
- Flutter
- Firebase
- Riverpod
- Hive
- Material 3
