# FOCUSORA

FocusOra is a study-focus application designed for students.

## MVP Features

- Pomodoro Timer
- Stopwatch Mode
- Focus Timer
- App Blocking
- Analytics Dashboard
- XP System
- Streaks
- Achievements

## Tech Stack

- Flutter
- Firebase
- Riverpod
- Hive
- Material 3

## Target Audience

- JEE Aspirants
- NEET Aspirants
- UPSC Aspirants
- College Students
