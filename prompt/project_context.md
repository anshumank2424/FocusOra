# FOCUSORA

Focusora is a study-focus application designed for students.

Target Users:
- JEE Aspirants
- NEET Aspirants
- UPSC Aspirants
- College Students

Core Features:
- Pomodoro Timer
- Stopwatch
- Focus Timer
- XP System
- Streaks
- Analytics Dashboard
- App Blocking

Future Features:
- Website Blocking
- YouTube Study Mode

Tech Stack:
- Flutter
- Firebase
- Riverpod
- Hive
- Material 3

Architecture:
- MVP Optimized Architecture
- Feature First Structure

Design:
- Dark Theme

- Modern Material 3
